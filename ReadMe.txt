

***************************************************************************************
**************************************  服务器端  *************************************
***************************************************************************************


**************************************** 必须 *****************************************

* lib				需要引入的依赖 jar

* axis2文件夹			搭建项目使用,将该文件夹下的所有内容copy到 %工程%/WebContent/WEB-INF/ 目录下

* axis2-web			搭建项目使用,将该文件夹下的所有内容copy到 %工程%/WebContent/ 目录下

* web.xml			用于与工程中的web.xml合并

* applicationcontext-webService.xml	
				放置于src目录下,并将文件中注释的引入行copy到 applicationcontext.xml 文件中


* demo文件夹			服务器端开发时,参考用 例子代码



*************************************** 选择使用 ***************************************

* Face.aar			部署项目使用,放置在 %Tomcat%/webapps/%工程%/WEB-INF/services 目录下 (开发时不需要理会)





***************************************************************************************
***************************************  客户端  **************************************
***************************************************************************************


*lib				需要引入的依赖 jar

*urls.properties		放置于src目录下





***************************************************************************************
***************************************  源码包  **************************************
***************************************************************************************

*src				源代码文件
